🐟 魚市場遊戲說明

完整的遊戲規則、策略指南與計算邏輯

🎯 遊戲目標

在魚市場遊戲中，您扮演魚貨批發商，需要從總代理手中進貨，再轉售給餐廳。您的目標是通過精明的買賣策略，在有限的資金下創造最高的投資報酬率（ROI）。

🏪 市場結構

🎣 供應端 - 總代理

  • 每日提供兩種魚類：A級魚、B級魚
  • 供應量有限，價高者得
  • 設有底價保護供應商權益

🏢 需求端 - 餐廳

  • 每日有固定預算採購魚貨
  • 價低者優先成交
  • 預算用完即停止採購

💼 您的角色 - 批發商

  • 向總代理買魚、賣給餐廳
  • 管理資金與庫存
  • 承擔滯銷風險

🏆 獲勝條件

投資報酬率（ROI）最高的團隊獲勝

ROI 計算公式：
ROI = (累積利潤 / 總投資金額) × 100%
總投資金額 = 初始資金 + 總借貸金額

💰 資金管理規則

初始資金

  • 每個團隊獲得相同的初始預算
  • 資金可用於購買魚貨

自動借貸機制

  • 當購買金額超過現有資金時，系統自動提供貸款
  • 貸款額度上限：初始預算的 50%
  • 每日按複利計算利息（通常為 3%/日）
  • 利息會自動從現金中扣除

🐟 魚貨交易規則

魚種分類

  • A級魚：高價值魚種，供應量較少，目標價格較高
  • B級魚：一般魚種，供應量較多，目標價格較低

投標限制

  • 每種魚最多可出 2 個不同價格的標單
  • 買入時：總出價不得超過 現金 + 剩餘借貸額度
  • 賣出時：出售數量不得超過庫存

成交規則

  • 買入：價格由高到低分配，直到供應量用完
  • 賣出：價格由低到高成交，直到餐廳預算用完
  • 固定滯銷：最高價標單的2.5%直接判定滯銷

📦 庫存管理規則

  • 每日結束後，所有庫存自動清零（模擬魚貨保鮮期限）
  • 未售出的魚貨收取滞銷費（通常 $10/kg）
  • 滯銷費會直接從現金中扣除
⚠️ 重要提醒：
  • 魚貨無法跨日保存，每日必須清倉
  • 滯銷費是固定成本，無法避免
  • 合理控制採購數量，避免過度庫存

🎮 操作規則

投標時間

  • 買入階段：通常 7 分鐘
  • 賣出階段：通常 4 分鐘
  • 時間到後系統自動結算，無法再修改標單

團隊協作

  • 每個團隊只能有一個人同時操作
  • 建議團隊內部先討論策略再投標
  • 可以查看市場資訊但無法看到其他團隊的標單

💡 買入策略

🎯 價格定位策略

  • 保守派：出價接近底價，確保低成本但可能得標量少
  • 積極派：出價較高確保得標，但壓縮利潤空間
  • 混合派：兩個標單設不同價格，分散風險

💰 資金管理策略

  • 善用借貸額度，但控制利息成本
  • 預留資金因應後續幾天的機會
  • 計算每日可承受的最大利息支出

📊 數量分配策略

  • 根據歷史銷售情況調整採購比例
  • A級魚利潤高但風險大，B級魚較穩定
  • 考慮餐廳預算限制，避免過度採購

🎯 賣出策略

⚠️ 滯銷風險控制

  • 避免最高價滯銷：不要盲目出最高價，有2.5%會直接滯銷
  • 價格梯度設計：用兩個價格測試市場接受度
  • 競爭對手分析：觀察其他團隊的定價傾向

💸 定價策略

  • 成本加成法：買價 + 目標利潤 + 風險緩衝
  • 市場導向法：參考目標價格設定合理區間
  • 競爭導向法：略低於預期的競爭對手價格
📈 價格設定建議：
  • 第一標：積極價格，確保部分成交
  • 第二標：保守價格，清倉止損
  • 務必考慮滯銷費成本 ($10/kg)

⚖️ 風險管理

💸 滞銷風險

  • 每公斤 $10 的處理費
  • 最高價 2.5% 固定滯銷
  • 餐廳預算不足導致的滯銷

📈 利息風險

  • 每日 3% 複利累積
  • 長期借貸成本驚人
  • 影響最終 ROI 計算

🎲 市場風險

  • 供需量每日變化
  • 競爭對手策略影響
  • 價格波動不確定性

🔢 買入階段計算

資金檢查邏輯：
總出價 = Σ(價格 × 數量)
可用資金 = 現金 + 剩餘借貸額度
if (總出價 > 可用資金) { 拒絕投標 }
成交分配邏輯：
1. 所有投標按價格降序排列
2. 從最高價開始，逐一分配供應量
3. 成交量 = min(投標量, 剩餘供應量)
4. 更新團隊庫存和現金/借貸
借貸計算邏輯：
if (成交金額 > 現金) {
  新借貸 = 成交金額 - 現金
  總借貸 += 新借貸
  現金 = 0
} else {
  現金 -= 成交金額
}

🔢 賣出階段計算

固定滞銷計算：
最高價投標 = 找出價格最高的所有投標
滯銷數量 = ceil(最高價投標總量 × 2.5%)
// 晚出價者優先滯銷
正常成交計算：
1. 扣除固定滯銷後，剩餘投標按價格升序排列
2. 從最低價開始，逐一購買
3. 成交條件：餐廳預算 ≥ 價格 × 數量
4. 預算不足時：部分成交 = floor(預算 ÷ 價格)
滯銷費計算：
總滞銷量 = 庫存 - 售出量
滯銷費 = 總滯銷量 × $10
現金 -= 滯銷費

🔢 每日結算計算

利息計算（複利）：
每日利息 = 總借貸 × 日利率(3%)
新總借貸 = 總借貸 + 每日利息
現金 -= 每日利息
if (現金 < 0) {
  追加借貸 = |現金|
  總借貸 += 追加借貸
  現金 = 0
}
每日損益計算：
每日收益 = 銷售收入 - 採購成本 - 滯銷費 - 利息
累積收益 += 每日收益
ROI 更新計算：
總投資 = 初始預算 + 總借貸本金
ROI = (累積收益 ÷ 總投資) × 100%
📊 範例1：買入投標與成交
市場狀況：A級魚供應量 1,200kg，底價 $120/kg
投標情況：
• 團隊甲：$200/kg × 500kg = $100,000
• 團隊乙：$180/kg × 600kg = $108,000
• 團隊丙：$160/kg × 400kg = $64,000
成交結果：
1. 團隊甲：$200/kg × 500kg → 成交 500kg
2. 團隊乙：$180/kg × 600kg → 成交 600kg
3. 團隊丙：$160/kg × 400kg → 成交 100kg（剩餘供應量）
💰 範例2：賣出投標與滯銷
市場狀況：A級魚餐廳預算 $300,000
投標情況：
• 團隊甲：$250/kg × 400kg（最高價）
• 團隊乙：$220/kg × 300kg
• 團隊丙：$200/kg × 500kg
固定滞銷：
最高價滞銷量 = ceil(400 × 2.5%) = 10kg
正常成交（按價格升序）：
1. 團隊丙：$200/kg × 500kg → 成交 500kg（花費 $100,000）
2. 團隊乙：$220/kg × 300kg → 成交 300kg（花費 $66,000）
3. 團隊甲：$250/kg × 390kg → 成交 390kg（花費 $97,500）
    剩餘預算：$300,000 - $163,500 = $136,500
滯銷處理：團隊甲 10kg × $10 = $100 滯銷費
📈 範例3：每日結算計算
團隊丙第3天結算：
• 買入成本：A級魚 200kg×$160 + B級魚 400kg×$90 = $68,000
• 銷售收入：A級魚 180kg×$200 + B級魚 350kg×$120 = $78,000
• 滞銷費：(20+50)kg×$10 = $700
• 前日借貸：$150,000，利息 = $150,000×3% = $4,500
當日損益：
當日利潤 = $78,000 - $68,000 - $700 - $4,500 = $4,800
更新借貸：總借貸 = $150,000 + $4,500 = $154,500
ROI更新：
假設累積利潤 = $25,000，初始預算 = $1,000,000
ROI = $25,000 ÷ ($1,000,000 + $154,500) × 100% = 2.16%

🎯 成功案例分析

保守穩健型策略

  • 買入時出價略高於底價，確保穩定供應
  • 賣出時定價合理，避開最高價滯銷風險
  • 控制借貸規模，降低利息負擔
  • 適合對象：求穩不求快的團隊

積極進取型策略

  • 善用借貸額度，擴大交易規模
  • 重點投資A級魚，追求高利潤
  • 靈活調整策略，快速適應市場
  • 適合對象：有經驗且願意承擔風險的團隊
